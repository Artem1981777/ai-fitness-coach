# 🦾 Iron Terminal Coach

**Iron Terminal Coach** — это интеллектуальный AI-агент, разработанный для хакатона **Cloudflare Agents Day**. Он помогает пользователям планировать тренировки и анализировать физическую форму прямо через терминал или веб-интерфейс, используя мощь Cloudflare Workers AI.

## 🚀 Технологический стек
* **Runtime:** Cloudflare Workers
* **AI Model:** @cf/meta/llama-3-8b-instruct (Cloudflare Workers AI)
* **Environment:** Android Termux (Development Environment)
* **Deployment:** GitHub Pages & Cloudflare Global Network

## 🛠 Функционал
- **AI Workout Generation:** Персонализированные планы тренировок на основе запросов пользователя.
- **Terminal First:** Оптимизировано для работы в консольном окружении.
- **On-chain Ready:** Архитектура подготовлена для интеграции с протоколами репутации (Talent Protocol).

## 📦 Как запустить (Development)
Так как проект разработан в среде Termux:
1. Клонируйте репозиторий.
2. Установите зависимости (если применимо).
3. Используйте `wrangler dev` для локального тестирования AI-функций.
